# MTG-Cataloguer
A work-in-progress GUI-based MTG card cataloging software that has offline support. <br/>

# Possible future features: <br/>
A main tab that holds the Check for Updates button, the Update JSON Prices button, the ability to start and stop a local webserver utilizing the qsettings JSON paths, total price calculation of collection, and file export button to export whole collection as a csv in a users chosen format. <br/>
An active progress bar for file downloads for the Check for Updates button.
